# CurseForge Description

This file keeps the current marketplace copy for `GBankManager` so future releases do not have to reconstruct it from chat history.

## Short Description

Guild Bank Manager is a World of Warcraft addon for guild officers who want one place to scan guild-bank inventory, track minimum stock, manage member requests, review guild-bank logs, and export shopping data.

## Full Description

Guild Bank Manager is a WoW guild-management addon built for keeping a guild bank organized, stocked, and accountable without juggling spreadsheets, chat notes, and manual bank checks.

It gives officers a full guild-bank workspace with:

- inventory scanning with per-tab stock visibility
- minimum stock management for important items
- member request submission and approval workflows
- export tools for shopping and restocking
- audit history and a dedicated Bank Ledger for item and gold activity
- dashboard reporting for shortages, withdrawals, and current stock readiness

### What It Does

#### Scan and Track Your Guild Bank

Open the guild bank and scan current inventory into the addon. GBankManager keeps tab-aware stock data so officers can see what is in the bank, where it lives, and what needs attention.

#### Manage Stock Minimums

Set minimum stock levels for the items your guild actually cares about. The addon highlights shortages, tracks restock targets, and helps convert approval activity into maintained stock rules.

#### Handle Member Requests

Members can submit stock requests through a guided request flow, and officers can review, approve, reject, or fulfill them from the main shell. Approved requests can also feed back into recurring stock rules.

#### Review Bank Activity in One Place

The `Bank Ledger` tab centralizes guild-bank item logs and money logs. Filter activity by action and date range, review deposits or withdrawals, export CSV, and see who is using the bank the most.

#### Export Shopping Data

Generate export output for:

- Auctionator
- TradeSkillMaster
- CSV
- Shopping List

This makes it easier to turn shortages and restock plans into an actual shopping pass.

### Highlights

- officer-first guild-bank dashboard
- inventory, minimums, requests, history, exports, and bank-ledger workspaces
- guild-bank log retention controls
- configurable critical-shortage threshold
- theme support and adjustable UI scale
- optional minimap launcher
- local test harness and active ongoing development

### Best Fit

GBankManager is best for guilds that want stronger structure around consumables, materials, crafted goods, restocking, and request handling, especially when multiple officers or crafters share responsibility for the bank.

### Current Status

The addon is under active development, with current work focused on refining the shell, ledger workflows, and broader guild-bank management experience while keeping the data model stable.
